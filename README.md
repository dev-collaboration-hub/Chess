# Lightweight Chess Game ♟️

A lightweight Chess game built from scratch with a focus on clean architecture, game logic, performance, and core programming fundamentals.

## 🎯 Project Goal

The goal of this project is to create a simple and efficient Chess implementation while understanding board game systems, rule engines, and decision-based gameplay.

## ✨ Features

* Complete chess board system
* Chess piece movement
* Legal move validation
* Turn management
* Capture system
* Check detection
* Checkmate detection
* Game state tracking
* Move history

## 🏗️ Architecture

The project focuses on separating important game components:

* Game Loop
* Board System
* Piece System
* Move Generator
* Rule Engine
* Validation System
* Game State Manager
* Rendering System
* Input System

## 🗺️ Roadmap

### M1 — Chess Foundation

* Setup project structure
* Create chess board
* Create piece representation
* Implement game state

### M2 — Core Chess Engine

* Add piece movement rules
* Generate legal moves
* Implement captures
* Add turn system

### M3 — Advanced Chess Rules

* Add check detection
* Add checkmate detection
* Add castling
* Add pawn promotion
* Add en passant

### M4 — Optimization & Release

* Improve engine performance
* Refactor architecture
* Add documentation
* Release stable version

## 🛠️ Tech Stack

* Python
* Minimal dependencies

## 📌 Vision

Build a lightweight Chess game that demonstrates strategic gameplay, rule-based systems, and clean engine architecture.
